# 小浦计算器 MVP 使用说明

## 功能概述

这是小浦计算器的MVP版本，实现了README.md中描述的核心功能。

## 已实现功能

### 1. 界面布局
- ✅ 上下中三层结构
- ✅ 下层：数字键盘（0-9、+、-、×、÷、=、C、⌫、.）
- ✅ 中层：当前输入显示区域
- ✅ 上层：历史计算记录显示

### 2. 历史记录功能
- ✅ 点击"="后自动保存计算历史
- ✅ 历史记录区域显示所有计算过程
- ✅ 清空按钮：清除所有历史记录
- ✅ 求和按钮：对历史记录中的所有结果求和

### 3. 菜单栏
- ✅ 左上角菜单按钮
- ✅ 设置选项（界面已准备，功能待完善）：
  - 声音模式
  - 皮肤颜色
  - 字体大小
  - 显示记录日期
  - 小数位数设置
  - 语言设置
- ✅ 登录按钮（界面已准备）

## 使用方法

### 基本计算
1. 点击数字按钮输入数字
2. 点击运算符按钮（+、-、×、÷）
3. 继续输入数字
4. 点击"="获得结果

### 历史记录操作
1. 计算完成后，结果会自动添加到历史记录
2. 点击"清空"按钮清除所有历史记录
3. 点击"求和"按钮对所有历史结果求和

### 其他操作
- "C"按钮：清除当前输入
- "⌫"按钮：删除最后一个字符
- 菜单按钮：展开/收起设置面板

## 运行应用

### Web版本
```bash

```

### 移动端（需要配置相应SDK）
```bash
flutter run
```
