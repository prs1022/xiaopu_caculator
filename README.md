# xiaopu_caculater

这个项目我想做一个小浦计算器，对标市场上的一些计算器。
我想做的功能粗略有以下几点：
1. 首先界面布局：上下中三层结构，下面是键盘部分，包含了数字、加减乘除等于号等，和常规的一样；中层比较宽的一行是输入的内容部分； 上层是在输入“=”号后 按行显示的历史计算内容；
2. 点击上层历史输入部分会触发选项卡，两条内容；1. 清空： 可以清掉上屏内容；2.求和：对上屏内容的每一行求和然后输出结果至中层
3. 计算器页面左上角放入菜单栏，添加以下几种设置：1. 声音模式： 滴滴声、哒哒声、还有播报数字的人声； 2.皮肤颜色（下层键盘部分设置、中层设置、上层设置）3.字体大小；4.显示记录日期；5. 小数位数设置 6. 语言设置
4. 在菜单栏的底部增加一个登陆按钮，方便登陆后可以开启全部特性，如果后续还有新特性需要开发的话
