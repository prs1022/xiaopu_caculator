# 音频文件说明

## 按键音效文件
- `beep.mp3` - 滴滴声音效
- `click.mp3` - 哒哒声音效

## 人声文件
### 数字
- `voice_0.mp3` - "零"
- `voice_1.mp3` - "一"
- `voice_2.mp3` - "二"
- `voice_3.mp3` - "三"
- `voice_4.mp3` - "四"
- `voice_5.mp3` - "五"
- `voice_6.mp3` - "六"
- `voice_7.mp3` - "七"
- `voice_8.mp3` - "八"
- `voice_9.mp3` - "九"

### 运算符
- `voice_plus.mp3` - "加"
- `voice_minus.mp3` - "减"
- `voice_multiply.mp3` - "乘"
- `voice_divide.mp3` - "除"
- `voice_equals.mp3` - "等于"
- `voice_dot.mp3` - "点"

### 功能键
- `voice_clear.mp3` - "清除"
- `voice_delete.mp3` - "删除"

## 文件格式要求
- 格式：MP3 或 WAV
- 时长：建议0.5-1秒
- 音质：清晰，音量适中
